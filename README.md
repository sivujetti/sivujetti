# Sivujetti

Sivujetti is a website builder and CMS.

## Requirements

- Apache or nginx
- Php 8.0.0 >= with
    - pdo_sqlite _or_ pdo_mysql
    - mbstring
    - sodium
    - fileinfo
    - xdebug (not required but recommended if you develop locally)
