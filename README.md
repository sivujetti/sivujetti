## Requirements

- Apache or nginx
- Php 8.1.0 >= with
    - pdo_sqlite _or_ pdo_mysql
    - mbstring
    - sodium
    - fileinfo
    - zip (required only if you want to install updates)
    - xdebug (not required but recommended if you develop locally)
